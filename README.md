# ShaBaas AI

Official public repository for ShaBaasPay AI tooling.

This repository contains:

- `@shabaaspay/agent-toolkit` (TypeScript) for integrating ShaBaasPay APIs with popular agent frameworks through function calling.
- OpenAPI artifacts used for public developer integration.

## Packages

- `tools/typescript/packages/agent-toolkit`

## Quick Start

```bash
cd tools/typescript
npm install
npm run build
```

Toolkit documentation:

- `tools/typescript/packages/agent-toolkit/README.md`
